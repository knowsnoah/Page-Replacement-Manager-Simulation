# Page-Replacement-Manager-Simulation
### Overview

This project simulates three page replacement algorithms used in operating systems:

First In First Out (FIFO)
Least Recently Used (LRU)
Optimal Page Replacement (OPT)

The goal of this project is to understand how these algorithms work and compare their performance based on the number of page faults. The simulation runs multiple tests using randomly generated reference strings and evaluates each algorithm under different page frame sizes.
